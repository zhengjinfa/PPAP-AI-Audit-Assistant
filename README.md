# PPAP-AI-Audit-Assistant
AI-powered PPAP audit tool that runs 100% locally. Supports offline rule-based audits and online deep audits via multiple LLMs. Multi‑language (中文/English/Deutsch). Zero data upload, privacy‑first.

An intelligent, privacy-first PPAP (Production Part Approval Process) audit assistant that runs entirely on your machine. It combines a built-in offline rule engine with the ability to invoke multiple large language models (DeepSeek, OpenAI, Ollama, etc.) for deep AI reviews – all without ever uploading your files to the cloud.

## Why This Tool?

Suppliers and quality engineers deal with dozens of Excel/CSV files per PPAP submission. Manual checking is time‑consuming and error‑prone. This tool automates the initial review and provides a detailed, actionable audit report in seconds.

- **🛡️ 100% Local & Secure** – All file parsing and analysis happen in your browser or Electron desktop app. No data leaves your computer. Encrypted local storage.
- **📋 Offline Rule Engine** – Instantly checks for missing documents (PSW, FMEA, Control Plan, MSA, etc.), inconsistent part numbers, abnormal CPK values, and outdated test reports.
- **🤖 AI Deep Audit** – One‑click switch between local offline audit and online AI review. Supports DeepSeek, OpenAI, Ollama, and any custom OpenAI‑compatible endpoint. Presets auto‑fill API URL and model name – no manual configuration needed.
- **🌐 Multi‑Language** – Full UI and report support for **Chinese (中文), English, and German (Deutsch)**. Perfect for international teams and global suppliers.
- **📊 Smart Report Rendering** – Markdown‑based audit report with color‑coded conclusion banner, non‑conformance cards, and SMART corrective action tables. Export to PDF with one click.
- **🔌 Flexible Model Switching** – Pre‑configured models with auto‑parameter filling. Supports custom model entry. API keys are encrypted and stored locally.
- **📁 Drag & Drop Simplicity** – Just drag your PPAP folder onto the interface. Supports `.xlsx`, `.xls`, and `.csv` files.

## Disclaimer

> ⚠️ The audit results generated by this tool are for reference only and **cannot be used as the final basis** for PPAP approval. Always verify findings with a qualified quality engineer.

## Quick Start (Web Version)

1. Clone this repository.
2. Open `index.html` in your browser.
3. Drag your PPAP folder into the drop zone.
4. Click **Offline Audit** (free, no setup) or configure a model for **AI Deep Audit**.

No server, no build step – it just works.
